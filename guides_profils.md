# Guide d'Utilisation par Profil - MRS Lubricants 👥

Ce document précise les responsabilités et les actions clés pour chaque rôle utilisateur au sein de la plateforme.

---

## 🎖️ PROFIL : ADMINISTRATEUR (ADMIN)
*Responsable de la gouvernance et de la sécurité.*

### Actions Clés :
- **Configuration Master** : Créer les utilisateurs, les produits et les zones.
- **Audit Forensique** : Surveiller le menu **Audit** pour tracer chaque action effectuée sur le système.
- **Paramétrage Financier** : Définir les objectifs commerciaux et les taux de commission.
- **Supervision Totale** : Accès à tous les dashboards (Finance, Stocks, Ventes).

---

## 📈 PROFIL : MANAGER / GÉRANT (MANAGER)
*Responsable de la validation opérationnelle et financière.*

### Actions Clés :
- **Validation Logistique** : Confirmer la réception des BL pour faire entrer le stock dans les dépôts partenaires.
- **Approbation des Ventes** : Examiner et valider les déclarations de vente faites par les commerciaux.
- **Gestion des Créances** : Suivre les factures impayées et enregistrer les encaissements.
- **Règlement Commissions** : Valider et marquer les commissions comme payées après vérification.
- **Alertes** : Suivre les notifications de stock bas pour réapprovisionner le Central.

---

## 💼 PROFIL : COMMERCIAL (COMMERCIAL)
*Responsable de la relation client et de la déclaration des ventes.*

### Actions Clés :
- **Prospection** : Créer de nouveaux **Partenaires** sur le terrain (Capture GPS automatique).
- **Déclaration de Vente** : Enregistrer les ventes effectuées à partir du stock déposé chez le partenaire.
- **Encaissement** : Déclarer les paiements reçus des clients pour les factures à crédit.
- **Suivi Performance** : Consulter son dashboard personnel pour voir l'atteinte de son objectif mensuel.

---

## 🚚 PROFIL : LIVREUR (DELIVERY)
*Responsable du transport physique du stock.*

### Actions Clés :
- **Consultation des BL** : Voir la liste des livraisons à effectuer vers les partenaires.
- **Confidentialité** : Pas d'accès aux prix de vente ni aux marges financières. Le livreur se concentre uniquement sur les **unités (bidons)**.
- **Justification** : Noter tout écart ou incident lors du transport via les commentaires de livraison.

---

## 🧱 RÈGLES COMMUNES À TOUS
1.  **Identification** : Toute action (modification, création) est enregistrée avec votre identifiant et votre adresse IP.
2.  **Transparence** : Les suppressions sont désactivées. Utilisez les justifications pour expliquer les erreurs de saisie.
3.  **Proactivité** : Consultez quotidiennement le centre de notifications (Cloche) pour les tâches urgentes.
